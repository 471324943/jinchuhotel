连锁酒店管理股份有限公司-公司简介


连锁酒店管理股份有限公司创立于2007年，独具资本IP，是国内首家全国性精品连锁酒店新三板挂牌企业（股票名称：股份，股票代码：833623），连续四年入选住宿和餐饮业新三板创新层唯一的酒店企业，首家入选“三板成指样本股”和“新三板引领指数创新成指样本股”的酒店企业。在行业中首创“事业合伙人”机制，鼓励员工与合作伙伴持有企业股份，做企业的主人，并已完成2轮企业员工股权激励。多次蝉联中国饭店金马奖“中国最具投资价值酒店品牌”，中国酒店业金光奖“中国酒店业中档酒店领军品牌”等荣誉。 酒店集团通过投资、兴建、合作等形式，专注于酒店连锁运营、设计、施工、培训、管理IT等服务，全面打造中高端连锁品牌酒店生态圈，为合作方提供安全、稳健、可靠的投资回报。截止目前，酒店已在北上广深，浙江、江苏、湖南、江西、河南、海南等省市拥有在营、筹建、签约及管理门店200余家。
      ，倡导“不将就”的生活态度，注重消费者体验，为认真生活的消费者提供不将就的生活空间，汇聚专业的研发优势不断升级，现已拥有国际酒店（盛酒店）、韵酒店、承酒店、宜酒店、wepai公寓、美隅民宿、奢缦、欣源、花生酒店等十大住宿业产品，并主打旗下六大品牌酒店，独创中华民族特色个性化感动服务，打造一个高品质生活体验馆，一家可玩、可乐、可拍的沉浸式体验连锁酒店。
